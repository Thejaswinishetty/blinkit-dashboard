# Blinkit Grocery Sales Analysis Dashboard

## Objective
Analyze Blinkit grocery sales data to understand sales trends, outlet performance, and delivery efficiency.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Analysis Performed
- Cleaned and transformed data using Power Query
- Created KPIs using DAX
- Analyzed outlet-wise and category-wise sales
- Studied delivery time patterns

## Dashboard Insights
- Identified top-performing outlets
- Found high-contributing product categories
- Analyzed delivery performance trends

## Conclusion
This dashboard provides actionable insights to improve sales operations and delivery efficiency.
